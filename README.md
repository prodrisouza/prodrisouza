Engenheira de Dados Sênior com +5 anos em projetos end-to-end: desde a ingestão de APIs e sistemas legados até entrega de modelos dimensionais, dashboards e governança de dados em produção. Já atuei em **Globo, Itaú, Petlove, Claro, Drogaria Araújo, BASA, FMC** e outros, com projetos cobrindo os quatro principais ambientes cloud: AWS, GCP, Azure e OCI.
